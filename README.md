Multi-Modal HR Employee Culture Insight Analyzer

An AI-powered HR analytics platform that helps organizations understand and improve workplace culture using Computer Vision, Natural Language Processing (NLP), and Data Visualization.

📌 Overview

This project analyzes:

Employee event images
Employee feedback text
Workplace engagement metrics

to generate actionable insights for HR teams and management.

The system combines image-based emotion recognition, NLP sentiment analysis, and culture scoring into a unified dashboard.

🚀 Features

✅ Employee Participation Detection
✅ Emotion Recognition from Images
✅ NLP-Based Feedback Analysis
✅ Workplace Culture Scoring
✅ HR Analytics Dashboard
✅ Interactive Employee Interface

🎯 Objectives
Recognize participation and emotions from event images
Analyze employee feedback using NLP
Measure employee engagement and workplace culture
Visualize HR insights using dashboards
